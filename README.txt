This is sample Test NG + selenium project in Java. It shows how to create and delete hotel bookings.

Pre-requisities:
===============
Java
Eclipse
TestNG
ChromeDriver

Install chromedriver
====================

You need to install chromedriver to run test on Google Chrome. 
Then set chromedriver path using final variable CHROMEDRIVER_PATH of src/main/java/com/hotelbooking/Booking.java class.
